# DNA-_Variant_protein

DNA 서열을 보고, 서열의 유전적 변이가 단백질의 구조적/ 기능적 변화로 이어지는 과정을 분석하는 프로그램을 (웹 형식으로 쓰기 쉽게)만들어줘

# 분석 파이프라인
## 1. 염기 서열 분석
## 2. 변이 지점 식별
## 3. 물리화학적 특성 분석(bio-python을 사용해 단백질의 분자량, 등전점, 불안정 지수, 지방족 지수, 친수성 지수 등을 계산    
## 4. 기능적 영향 예측(Meta AI의 ESM-2 언어 모델을사용해 치명적인 강도를 예측해줘)
